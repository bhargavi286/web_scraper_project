Web Scraper Project
Description

This project is a Python based web scraper that collects data from websites and saves it into a CSV file. It also includes a Streamlit dashboard to view and analyze the scraped data.

Features

Scrapes data from website

Saves data into CSV file

Multi page scraping support

Streamlit dashboard for viewing data

Search option in dashboard

Basic data visualization

Technologies Used

Python

Requests

BeautifulSoup

Pandas

Streamlit

Matplotlib

Files

scraper.py

dashboard.py

data.csv

requirements.txt

How to Run

Install dependencies
pip install -r requirements.txt

Run scraper
python scraper.py

Run dashboard
streamlit run dashboard.py

Output

Scraped data stored in CSV

Data shown in dashboard

Search and simple graphs available
